## TypeSCript

### Interface vs Type

`Type`은 새 프로퍼티를 추가하도록 개방될 수 없는 반면, `Interface`의 경우 항상 확장될 수 있다.

### 타입 추론

- 타입스크립트가 알아서 타입을 추론
- 모든 것을 다 쓸 필요는 없음

### Union Type vs Intersection Type

`Union Type`: 여러 타입 중 하나
`Intersection`: 교집합, 모두를 만족하는 타입

### Optional Parameter

해당 Parameter가 있을 수 도 있고, 없을 수도 있을 때 사용
